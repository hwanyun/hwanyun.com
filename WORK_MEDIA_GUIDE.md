# 작품 상세 페이지 미디어 배열 규칙

모든 작품 페이지는 12열 그리드를 사용한다. 사진과 영상을 같은 크기의 카드로 반복하지 않고, 작품을 읽는 순서를 만든다.

## 기본 위계

| 역할 | 그리드 폭 | 사용 기준 |
| --- | --- | --- |
| `hero` | 9열 | 각 페이지의 첫 번째 이미지 또는 영상. 작품의 첫인상을 담당한다. |
| `wide` | 7열 | 설치 전경, 핵심 장면, 주요 영상처럼 내용을 이어가는 자료. |
| `standard` | 5열 | 보조 설치 전경, 관객과의 상호작용, 맥락을 보여 주는 자료. |
| `detail` / `portrait` | 4열 | 장치의 디테일, 세부 사진, 세로 영상. 4열보다 작게 만들지 않는다. |

`align: left`와 `align: right`를 사용해 여백의 위치를 번갈아 조절한다. 같은 역할의 자료가 계속 이어질 때도 좌우 정렬을 교대해 리듬을 만든다.

## 자동 적용 규칙

미디어에 `layout`을 직접 쓰지 않으면 페이지는 다음 순서로 배치한다.

1. 첫 번째 미디어는 `hero`
2. `width: full`은 `wide`
3. `width: third`는 `detail`
4. 세로 비율(`ratio: 9:16`)의 영상은 `portrait`
5. 그 밖의 미디어는 `standard`

특정 작품의 서사가 더 강하게 필요할 때만 frontmatter에 `layout`과 `align`을 직접 지정한다. 현재 `coexistence`와 `intuition-i`는 이 방식으로 개별 아트디렉션을 적용했다.

## 편집 원칙

- 모든 자료를 동등하게 크게 놓지 않는다.
- 대표 이미지 다음에는 넓은 주요 기록을 배치하고, 작은 자료는 전환과 디테일을 위해 사용한다.
- 작은 사진과 세로 영상도 데스크톱에서 최소 4열을 유지해 내용을 식별할 수 있어야 한다.
- 모바일에서는 모든 미디어를 한 열로 표시해 가독성을 우선한다.
