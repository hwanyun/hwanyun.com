# 연작(Series) 연결 규칙

연작 작품은 상세 페이지 상단에 `PART OF THE SERIES`를 표시하고, 같은 연작의 다른 작품으로 바로 이동할 수 있다. 링크 목록은 자동으로 만들어지므로 개별 작품의 URL을 서로 수동으로 적지 않는다.

## 적용 기준

- 제목에 I, II, III 등 번호가 붙더라도 작품 간에 하나의 연작 관계가 확인될 때만 사용한다.
- 소재나 제작 시기가 비슷하다는 이유만으로 연결하지 않는다.
- 현재 적용된 연작은 `Intuition: Untitled I–II`, `Three Pieces: Untitled I–III`이다.

## 작품 파일에 추가할 정보

같은 연작의 **모든** `src/content/works/*.md` 파일에, 완전히 같은 `id`와 `label`을 넣는다.

```yaml
series:
  id: "intuition-untitled"
  label: "Intuition: Untitled"
```

- `id`: 연작을 묶는 내부 식별자. 영문 소문자와 하이픈을 사용한다. 예: `three-pieces-untitled`.
- `label`: 화면에는 나타나지 않는 그룹명 대신, 페이지 접근성 정보와 관리 기준으로 사용하는 연작명이다. 연작 전체에서 동일하게 유지한다.
- 페이지에는 해당 작품을 포함한 연작의 모든 멤버가 제목순이 아니라 `order` 순으로 표시된다. 현재 보고 있는 작품은 링크가 아닌 현재 항목으로 표시된다.

## 새 작품을 연작에 추가할 때

1. 기존 연작에 속하면 그 연작의 `id`와 `label`을 그대로 복사한다.
2. 새 연작이면 첫 작품부터 위 `series` 블록을 추가하고, 이후 모든 멤버에 같은 값을 넣는다.
3. `npm run build`로 페이지 생성과 링크를 확인한다.
4. 관리자 화면에서 추가할 때도 **연작 연결 (Series)**의 두 값을 동일하게 입력한다.

한 작품에만 `series`를 넣으면 연작 링크는 표시되지 않는다. 두 작품 이상이 같은 `id`를 공유해야 한다.
